# Memory - Jeu de l'Alphabet Français

Un jeu de memory interactif pour apprendre l'alphabet français avec un design minimaliste géométrique moderne.

## Configuration GitHub Pages

Si vous voyez une erreur 404, veuillez configurer GitHub Pages manuellement :

1. Allez sur https://github.com/luiscoixao/Jeux
2. Cliquez sur **Settings** (Paramètres)
3. Dans le menu de gauche, cliquez sur **Pages**
4. Sous "Build and deployment" → "Source", sélectionnez **Deploy from a branch**
5. Sélectionnez la branche **main** et le dossier **/ (root)**
6. Cliquez sur **Save**

GitHub Pages devrait maintenant déployer le site. Attendez 1-2 minutes, puis accédez à :
**https://luiscoixao.github.io/Jeux/**

## Caractéristiques du jeu

- Grille 4×4 avec 8 paires de lettres (A-H)
- Design minimaliste avec palette bleu électrique et rose vif
- Flip fluide des cartes
- Compteur de coups et de paires trouvées
- Message de victoire
- Responsive design (fonctionne sur mobile et desktop)
- Compatible avec tous les navigateurs (Chrome, Firefox, Safari, Edge)

## Technologies utilisées

- React 19
- Tailwind CSS 4
- Vite (compilé en fichiers statiques)
- TypeScript
